如何搭建个人博客

Veröffentlicht am 2018-08-02

简单几步搭建一个个人博客，首先是效果：https://jdroida.github.io/

然后开始自己动手吧

  1. 首先创建一个github账号。网址：https://github.com/。然后创建一个仓库，如图：![创建git仓库](https://muchat2017.oss-cn-shanghai.aliyuncs.com/%E5%88%9B%E5%BB%BAgit%E4%BB%93%E5%BA%93.png)

    仓库名输入username.github.io 。比如我的用户名叫YURAO1120，名字就叫YURAO1120.github.io
  2. 安装相关软件，包括git，node，npm（在windows下node包括了npm），hexo

    Node官网：https://nodejs.org/en/；

    Git官网：https://git-scm.com/downloads

    安装完了之后在cmd或者git bash输入这些指令，能看到版本号就算成功了。

    另外在任意地方鼠标右键就能看到git bash。Git bash能使用几个最基本的linux指令，比如ls，pwd这些。然并卵。Hexo是最后安装的，在gitbash执行npm install hexo-cli
  3. 在你准备编写博客的目录下，用gitbash输入hexo init username,github,io
    Username是你的github用户名，然后就会生成一个文件夹。然后在username.github.io/_config.yml中修改几个键值对：
    title: 软萌小屋//你博客的名字
    author: YURAO //你的名字
    language: zh-Hans //语言 中文
    deploy:
    type: git //使用Git 发布
    repo: https://github.com/username/username.github.io.git // 刚创建的Github仓库
  4. 安装自动部署发布工具：在username.github.io文件夹下，打开gitbash，输入：npm install hexo-deployer-git –save
  5. 最后就可以测试发布了。还是在上一步的gitbash中，输入：hexo clean && hexo g && hexo d 。这个过程中 可能会让你输入github的账号和 密码。 然后就可以打开username.github.io了
  6. 当需要编写新的博客的时候，在username.github.io文件夹下面，打开gitbash，输入hexo new 日志标题。然后在username.github.io/source/_posts文件夹下面会生成一个 MD文件，然后编辑这个文件，在这个文件里面写日志，写完之后哦就可以发布了。发布的方法还是在username.github.io文件夹下打开gitbash，输入hexo clean && hexo g && hexo d

写完之后可以看到和举例的博客还有蛮大差别，这里进行一些优化。

  1. 增加next主题。next是hexo框架下最火热的主题，也是我的个人博客使用的主题。关于hexo其他框架可以自行在google进行搜索。首先进入username.github.io文件夹下，然后打开gitbash，输入git clone https://github.com/iissnan/hexo-theme-next themes/next然后打开username.github.io/_config.yml文件将theme设置为next。最重要的一点是看控制台！实际动手操作的时候我发现clone的 速度很慢，这时候我们需要手动下载源码进行操作。https://github.com/theme-next/hexo-theme-next/releases这个是最新版next的源码，我是实在拉不动了，直接下的源码，将下好的源码解压缩到username.github.io/themes/，文件夹取名叫next，然后修改_config.yml的theme属性即可。
  2. 为next主题添加动画。这里我添加的是一个2D动画，想了解3D动画或者各种其他动画的自行google，谢谢。打开上一步粘贴的next主题文件夹，在里面找到_config.yml,将canvas_nest属性设置为true即可。
  3. 开启打赏功能。默认支持支付宝、微信、比特币支付。在这里 我们开启支付宝和微信支付。首先存储支付宝和微信的收款码，将其存放在这个路径下：username.github.io/themes/next/source/images。然后打开上一步的_config.yml，在wechatpay后面加上wechatpay.jpg
    alipay后面加上alipay.jpg。另外#是注释的意思，这两行不要给注释掉了，记得删掉#。
